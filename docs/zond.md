# Zond (PoS / EVM Test Network)

QRL develops **Zond**, described in official documentation as a **Proof-of-Stake** network with **EVM-compatible smart contracts**, distinct from the classic XMSS + PoW mainnet stack.

## Public testnet

The documentation hub links to:

- **Zond testnet**: [test-zond.theqrl.org](https://test-zond.theqrl.org/)

Use that entry point for current wallet/explorer endpoints, faucet rules, and developer tooling for the Zond stack.

## Relationship to this hub

Most pages in **QRL Hub** describe the **legacy/mainline QRL ledger** (XMSS signatures, RandomX PoW, `qrl` node). Zond introduces a different execution and consensus model (PoS + EVM). When integrating, **confirm which network** your addresses and transactions target.

## Where to watch for updates

- [QRL Docs](https://docs.theqrl.org/) — top-level announcements and navigation
- [theQRL GitHub](https://github.com/theQRL) — node and tooling repositories
